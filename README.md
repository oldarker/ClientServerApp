ClientServerApp

Клиент-серверное приложение для вычисления суммы нечётных элементов матрицы

Приложение состоит из трёх частей:
- Клиент (Swing) — пользователь вводит размеры матрицы, отправляет её на сервер и получает результат
- Последовательный сервер — обрабатывает запросы клиентов по очереди (однопоточный режим)
- Параллельный сервер — обрабатывает запросы нескольких клиентов одновременно (многопоточный режим)

Технологии
- Java
- Swing — графический интерфейс клиента
- Сокеты — сетевое взаимодействие
- Сериализация — передача объектов по сети
- Многопоточность — параллельная обработка запросов
- Stream API & try-with-resources — нововведения Java 7 и 8

Как запустить
1. Запустите сервер (последовательный или параллельный)
2. Запустите клиент
3. Введите размеры матрицы → Отправить
